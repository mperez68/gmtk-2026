# Enter Input
Enter Input is a 16pt pixel art font designed to mimic keyboard input, mouse input, and controller input.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 16px
- Scales: 16, 32, 48, 64, etc
- Ascender Max: 0px
- Descender Max: 0px

## Kerning
None
