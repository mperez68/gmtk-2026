# Micro Chat
Micro Chat is a 5pt pixel art font designed to be as compact as possible while still being stylish and legible.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 5px
- Scales: 5, 10, 15, 20, etc
- Ascender Max: 3px
- Descender Max: 2px
