# Simple Script
Simple Script is a 7pt pixel font designed with simple clear lettering.

## Author
[VEXED](https://v3x3d.itch.io/)

## Basic Info
- Base Glyph Height: 7px
- Scales: 7, 14, 21, 28, etc
- Ascender Max: 4px
- Descender Max: 3px
